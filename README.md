# Проект: Место
### Описание
Интерактивная страница, где можно редактировать свой профиль, добавлять фотографии, удалять их и ставить лайки.


### Технологии:
+ HTML5
+ CCS
+ JavaScript
+ Адаптивная верстка (десктопы, планшеты, мобильные устройства)
+ Flexbox
+ Grid Layout
+ Позиционирование
+ Медиазапросы
+ Валидация форм
+ Рефакторинг кода (Проект 7)
+ Классы (Проект 7)
