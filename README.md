# Проект: Место
### Описание
Интерактивная страница, куда можно добавлять фотографии, удалять их и ставить лайки.


### Технологии:
+ HTML5
+ CCS
+ JavaScript
+ Адаптивная верстка (десктопы, планшеты, мобильные устройства)
+ Flexbox
+ Grid Layout
+ Позиционирование
+ Медиазапросы
